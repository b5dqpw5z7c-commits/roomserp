# Next Steps

Bu v2 paket canlı MVP için daha güvenli bir temel sağlar. Yine de aşağıdaki işler gerçek kullanım öncesi/sonrası önceliklidir:

1. Kapsamlı entegrasyon testleri: stok, satın alma, cari, kasa, bordro, QR.
2. Gerçek kullanıcı yetki matrisi: hangi rol hangi hareketi yapabilir.
3. Satın alma/satış belge numaralandırma serileri.
4. İptal/ters kayıt ekranları ve yetkileri.
5. Cari mahsup ekranı: avansların hangi satış/alış belgesine kapandığını kullanıcı onayıyla eşleştirme.
6. Backup otomasyonu ve düzenli geri yükleme testi.
7. Frontend UI'nın production düzeyinde yeniden yazılması.
